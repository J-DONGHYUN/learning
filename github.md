# GitHub

## GitHub란?

GitHub는 Git을 기반으로 하는 원격 저장소 서비스이다.

코드를 저장하고 관리하며 여러 개발자가 협업할 수 있도록 도와준다.

---

## GitHub의 주요 기능

### Repository
프로젝트 파일과 변경 이력을 저장하는 공간이다.

### Issue
프로젝트의 문제나 할 일을 기록하는 기능이다.

### Pull Request
브랜치에서 작업한 내용을 다른 브랜치에 병합하기 전에 검토하는 기능이다.

### Commit History
코드 변경 기록을 확인할 수 있다.

---

## GitHub 활용

GitHub는 코드 저장뿐만 아니라  
학습 내용을 기록하는 저장소로도 활용할 수 있다.
